## Packet Tracer Project

This repository contains a Cisco Packet Tracer (.pkt) file for an enterprise WAN design.

### How to Open
1. Download the `.pkt` file
2. Open it using **Cisco Packet Tracer (v8.x recommended)**

### Features
- Multi-protocol routing (OSPF, EIGRP, RIP)
- VLSM-based IP addressing
- NAT / PAT
- Extended ACLs
- DHCP with Relay Agents
